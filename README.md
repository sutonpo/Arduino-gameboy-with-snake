# Arduino-gameboy-with-snake

這是一個使用 **Arduino 和 LCD12864（ST7920 控制器） 的專案。

## 🔧 使用元件

- Arduino UNO
- LCD12864 (SPI 模式)
- 10k Potentiometer
- Breadboard and jump wires

🖼️ 專案展示

![LCD 顯示畫面](images/demo.jpg)


## 📜 功能說明

- 顯示貪食蛇遊戲畫面
- 支援遊戲選單
- 調整亮度 via potentiometer

## 🧠 使用的函式庫

- `U8g2` - 用於簡單顯示圖文資訊（支援 ST7920 控制器）
- `SPI.h`
